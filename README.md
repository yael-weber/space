# Space Survivor 🚀

משחק חלל מהנה וממכר שנבנה ב-JavaScript, HTML ו-CSS. אספו כמה שיותר יהלומים, הימנעו מהמוקשים, ועלו בשלבים אינסופיים!

## 🎮 איך משחקים?
* **תנועה:** השתמשו בעכבר כדי להזיז את החללית שלכם.
* **המטרה:** אספו את היהלומים הצהובים (💎) כדי לצבור נקודות.
* **זהירות:** אל תיגעו במוקשים האדומים (☄️) – נגיעה אחת והמשחק נגמר!
* **שלבים:** ככל שתצברו יותר נקודות, תעלו לשלבים מתקדמים יותר שבהם המשחק הופך למהיר ומאתגר יותר.

## 🛠️ טכנולוגיות
המשחק נבנה ללא ספריות חיצוניות (Vanilla JavaScript), מה שהופך אותו למהיר מאוד וקל להרצה בכל דפדפן.
